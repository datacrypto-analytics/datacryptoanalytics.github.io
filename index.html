Somos o primeiro fundo focado em Criptomoedas e Forex, nossos Traders operam dia e noite junto com algoritmos Machine learning obtendo pontos chave para negociações rentáveis

A negociação de produtos liquidados com Bitcoin não entregues é altamente especulativa e é adequada apenas para os Clientes que entendem e estão dispostos a assumir os riscos econômicos, legais e outros envolvidos, e são financeiramente capazes de assumir perdas significativamente superiores a Margem ou depósitos.

Além de oferecermos este novo serviço no Brasil e no mundo todo, criamos um projeto para ajudar familias com baixa renda passando necessidade, contribuimos com alimentos e produtos essenciais para a saúde e bem estar, livros para crianças entre outros meios de educação.

É um sistema de negociação algorítmica escrito para a plataforma de negociação MetaTrader que executa as ordens nas condições do mercado e foi criado para observar. Os desenvolvedores dos Expert Advisors usam a linguagem de programação MQL e aplicação de codificação MetaEditor.

Nossos algoritmos foram desenvolvidos com as bibliotecas mais conhecidas pela comunidade Machine Learning, usamos Sklearn, Matplotlib, TensorFlow, entre outras bibliotecas de fácil uso. Fazendo o downloads do anaconda3 já é possível rodar nossos algoritmos.

São bibliotecas de código aberto compatível com Python para computação numérica que torna o machine learning mais rápido e fácil.

Anaconda3 é uma distribuição gratuita e de código aberto das linguagens de programação Python e R para computação científica, que visa simplificar o gerenciamento e a implantação de pacotes. As versões do pacote são gerenciadas pelo sistema de gerenciamento de pacotes conda.

Se você não conseguiu instalar algma biblioteca ou rodar nossos algoritmos, ou achou algum bug, entre em nossa comunidade no Telegram e entraremos em contato.

Este projeto teve inicio com a idéia de usar o potencial do machine learning para criar um modelo de previsões de séries temporais.
